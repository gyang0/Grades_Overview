# Grades_Overview
Something to help me keep track of my academic scores.

Features:
- Adding courses
- Deleting courses
- Overview table for different grades

<br />

To compile:
1. Navigate to folder
2. <code>g++ main.cpp funcs.cpp course.cpp -o outputfile.exe</code>
3. <code>outputfile.exe</code>
